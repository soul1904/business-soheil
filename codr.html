Project Description

Codr is a website that allows users to easily design and create a website using our templates and instead of paying a monthly fee or such like other user-friendly website designer websites, gives the user the source code for the website with implemented comments that goes more in depth into the workings of the code the user designed. This allows the user freedom to do many things such as deploying the website with a domain the user purchased.we also implemented many TollTips through out the templates to serve as a educational/guiding tool through out their experience. This website appeals to many from those who simply want a simple website without paying an outside company fees to do so, to new web devs looking to have a finished product to modify and see the inner workings of the code.
